# kara-moon

Kara Moon - website pakket

Bestanden in deze map:
- index.html              De portfolio-website (de startpagina). Dit is de pagina die je gaat bouwen.
- kara_moon_funnel.html   De funnelpagina van de cursus. De knoppen op de website verwijzen hiernaartoe.
- kara_moon_vervolg.html  De vervolgpagina met Breakthrough+ en All-in, voor wie al binnen is.

Hoe gebruik je dit:
- Wil je het lokaal bekijken: dubbelklik op index.html, hij opent in je browser.
- Wil je het online zetten: upload deze drie bestanden samen naar je host of GitHub Pages.
  Houd de bestandsnamen gelijk, anders werken de knoppen tussen de pagina's niet.

Goed om te weten:
- Alle foto's en stijl zitten in de bestanden zelf, je hoeft niets anders te uploaden.
- De website is vindbaar voor Google. De funnel en vervolgpagina staan bewust op niet-vindbaar.
- De knoppen naar het programma wijzen naar je systeme.io pagina's. Controleer die links voor je live gaat.
- De mockup naast het aanbod is een voorbeeld, dat vervang je later door een echte foto.
